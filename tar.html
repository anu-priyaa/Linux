"tar" command in LINUX

The tar (short for tape archive) command is a utility used to create, extract, and manipulate archive files. It collects multiple files and directories into a single file, often with optional compression, to facilitate backup, transfer, or storage.

Syntax:

tar [options] [archive_name] [file_or_directory]

where:

  • [options] specifies what action to perform and any additional flags to modify behavior.
  • [archive_name] specifies the name of the archive file (e.g., archive.tar, backup.tar.gz).
  • [file_or_directory] specifies the files or directories to include in the archive or extract from it.

Purpose:

  • Data Archival: Combines multiple files into a single archive file for easy management.
  • Backup Creation: Saves copies of files and directories for restoration if data is lost.
  • File Transfer: Simplifies sharing and transferring multiple files over a network.
  • Space Saving: Compresses data to reduce disk usage.
  • Preservation of File Metadata: Retains file permissions, ownership, and timestamps.

Commonly used options

Option Description
-c Create a new archive.
-x Extract files from an archive.
-v Display verbose output (show files being processed).
-f Specify the archive file name.
-z Compress the archive with gzip.
-j Compress the archive with bzip2.
-t List the contents of an archive
