"gzip" command in LINUX

The gzip command in Linux is a file compression utility that reduces the size of files using the GNU zip algorithm. Compressed files are easier to store and transfer, and their names typically include a .gz extension.

Syntax:

gzip [options] [file]

where:

  • [options] : Optional flags that modify the behavior of the command.
  • [file] :The name of the file you want to compress or decompress. This can be a single file (e.g., file.txt) or multiple files (e.g., file1.txt file2.txt).

Purpose:

  • File Compression: Reduces the size of files to save disk space.
  • Efficient File Transfers: Smaller file sizes make uploading and downloading over a network faster.
  • Backup and Archiving: Often used alongside tar to compress archived data.
  • Ease of Use: Simple syntax and fast compression compared to other tools.

Applications:

Some applications of "gzip" command are:

  1. Compressing Files
    • Compress a single file to reduce its size

    • Decompress a .gz file to restore the original

Commonly used options

Option Description
-d Decompress a file.
-k Keep the original file after compression.
-v Show detailed output (compression ratio and size).
-r Compress all files in a directory recursively.
-t Test the integrity of a compressed file.
-# Set compression level (1 = fastest, 9 = best compression).
